Slideshow Documentation

Quick Start Guide

Getting Started:

  1. Select a Google Drive folder from the browser
  2. Click the green "Start Slideshow" button
  3. Your slideshow will begin automatically
  4. Press Escape to exit slideshow and return to folder selection
Note: The slideshow will automatically refresh content every few minutes and continue running indefinitely - perfect for TV displays.
Control Slide Duration

Add a duration prefix to filenames to control how long each slide displays:

[30]my-image.jpg → Displays for 30 seconds [5]quick-message.png → Displays for 5 seconds my-regular-image.jpg → Uses default duration (10 seconds)
Example: To show a sponsor logo for 15 seconds, rename the file to:
[15]sponsor-logo.png
Note: Videos always play for their full length, regardless of duration prefix.
Add Titles & Descriptions

Add text overlays by editing the file's description in Google Drive:

  1. Right-click a file in Google Drive and select "File information"
  2. Add text to the "Description" field
  3. The slideshow will display it as an overlay
Single paragraph: Shows as description only Multiple paragraphs: First = title, rest = description No description: No overlays (clean display)
Create Leaderboards

Add live competition leaderboards to your slideshow:

  1. Create a file with "leaderboard" in the filename (e.g., leaderboard_speed.md)
  2. Configure leaderboard settings in the file's description in Google Drive
  3. Leaderboards automatically pull live data from your competition system
Example Configuration (in file description):
title: Speed Championship
route_type: Official Speed
limit: 12
Tip: Add background images with "leaderboard_bg" in the filename (e.g., leaderboard_bg1.jpg) for dynamic backgrounds.
Background Images for Leaderboards

Enhance leaderboards with dynamic background images:

  • Add images with leaderboard_bg in the filename
  • Examples: leaderboard_bg1.jpg, leaderboard_bg_action.png
  • Backgrounds alternate left/right placement automatically
  • Multiple backgrounds cycle through slides
Tip: Use action shots or climbing wall photos for engaging backgrounds.
Global Settings

Configure slideshow behavior by creating a settings.txt file in your folder:

default_slide_duration_seconds: 8 slideshow_title: My Competition slideshow_auto_reload_minutes: 10 default_leaderboard_row_limit: 12

Add these settings to the file's description in Google Drive (not file content).

Keyboard Shortcuts

Control the slideshow with these keyboard shortcuts:

  • Escape - Exit slideshow and return to folder browser
  • F11 - Toggle fullscreen mode
  • → or Space - Next slide (manual control)
  • ← - Previous slide (manual control)
File Organization

Important: Files display in alphabetical order by filename.

Example order:
1. aaa-intro-image.jpg
2. bbb-video.mp4
3. ccc-leaderboard_speed.md
4. ddd-photo.jpg
Tip: Use prefixes like "001-", "002-", etc. to control the order of your slides.
Troubleshooting

Common Issues:

  • Folder not accessible: Make sure you have view permissions for the folder
  • Files not showing: Check that files are actually in the folder (not subfolders)
  • Leaderboard not updating: Verify the data endpoint URL in settings
  • Duration prefix not working: Ensure format is exactly [number]filename
Note: Videos must be in a Google Drive compatible format (MP4, MOV, etc.)
